# Ресурси для інсталятора

Покладіть сюди файли, які будуть вбудовані в інсталятор:

## wireguard-amd64.msi
Завантажте офіційний WireGuard MSI з https://download.wireguard.com/windows-client/
Перейменуйте у `wireguard-amd64.msi` та покладіть у цю папку.

## default.conf (опціонально)
Якщо потрібно вбудувати конфігурацію за замовчуванням — покладіть .conf файл 
сюди з назвою `default.conf`. Він буде автоматично імпортований під час інсталяції.
